## Descripción
Shell educativa en C con comandos básicos y avanzados.

## Comandos implementados
Basicos:
- listar
- leer <archivo>
- tiempo
- calc <a> <op> <b>

- salir

Avanzados:
- historial
- limpiar
- usuario
- directorio

## Compilacion y ejecucion

make run
